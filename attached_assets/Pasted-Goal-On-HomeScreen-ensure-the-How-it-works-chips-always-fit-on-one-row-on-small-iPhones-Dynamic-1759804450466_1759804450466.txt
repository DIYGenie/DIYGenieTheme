Goal: On HomeScreen, ensure the “How it works” chips always fit on one row on small iPhones. Dynamically set the second chip label to “Photo / Room scan” when space allows, otherwise “Room scan”. Keep the tiny chevron arrows. No horizontal scrolling.

File: app/screens/HomeScreen.*

Compute dynamic chip width & narrow flag

import { Dimensions } from 'react-native';

const screenW = Dimensions.get('window').width;
// total gutters: side padding ~24*2, gaps between 4 chips (~8*3), arrows (~10*3)
const arrowsW = 10 * 3;
const gapsW   = 8  * 3;
const sidePad = 24 * 2;
const chipWidth = Math.max(74, Math.floor((screenW - arrowsW - gapsW - sidePad) / 4));
const isNarrow = screenW < 390; // iPhone 12/13 mini, SE, etc.


Items array (dynamic label for chip #2)

const items = [
  { id: 1, icon: 'create-outline',   label: 'Describe' },
  { id: 2, icon: 'image-outline',    label: isNarrow ? 'Room scan' : 'Photo / Room scan' },
  { id: 3, icon: 'sparkles-outline', label: 'AI preview' },
  { id: 4, icon: 'list-outline',     label: 'Build plan' },
];


Render row (unchanged logic, just use dynamic chipWidth)

<View style={chipStyles.row}>
  {items.map((item, idx) => (
    <React.Fragment key={item.id}>
      <TouchableOpacity style={[chipStyles.chip, { width: chipWidth }]} activeOpacity={0.85}>
        <Ionicons name={item.icon as any} size={18} color={colors.brand} />
        <Text style={[chipStyles.chipLabel, { maxWidth: chipWidth - 12 }]} numberOfLines={1}>
          {item.label}
        </Text>
      </TouchableOpacity>

      {idx < items.length - 1 && (
        <View pointerEvents="none" style={chipStyles.arrowWrap}>
          <Ionicons name="chevron-forward" size={12} color="rgba(110,64,255,0.45)" />
        </View>
      )}
    </React.Fragment>
  ))}
</View>


Tighten styles

const chipStyles = StyleSheet.create({
  row: { flexDirection: 'row', alignItems: 'center', justifyContent: 'space-between' },
  chip: {
    height: 64, borderRadius: 12,
    backgroundColor: colors.brand50, borderWidth: 1,
    borderColor: 'rgba(110,64,255,0.12)',
    alignItems: 'center', justifyContent: 'center', paddingVertical: 8, paddingHorizontal: 6,
  },
  chipLabel: { fontSize: 10, lineHeight: 13, fontWeight: '600', color: colors.ink700, marginTop: 6, textAlign: 'center' },
  arrowWrap: { width: 10, alignItems: 'center', justifyContent: 'center', marginHorizontal: 4 },
});


Acceptance

On iPhone mini widths, the second chip shows “Room scan”; on larger widths it shows “Photo / Room scan”.

All four chips + chevrons fit on one row without clipping or horizontal scroll.

No new lint/TS errors; Home still renders instantly.
