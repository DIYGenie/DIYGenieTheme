New Project — center the bottom buttons above the tab bar

Goal: Place Scan Room and Upload Photo in a sticky, centered row above the bottom tab bar. Keep them disabled until the form is valid.

Use safe area + real tab bar height

In NewProjectForm.tsx, import:

import { useSafeAreaInsets } from 'react-native-safe-area-context';
import { useBottomTabBarHeight } from '@react-navigation/bottom-tabs';


Inside the component:

const insets = useSafeAreaInsets();
const tabBarHeight = useBottomTabBarHeight(); // accurate height
