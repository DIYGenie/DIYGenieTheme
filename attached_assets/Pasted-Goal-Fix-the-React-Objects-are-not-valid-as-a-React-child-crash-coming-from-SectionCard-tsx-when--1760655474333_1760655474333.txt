At the top, ensure:

import React from 'react';


In props, make sure:

Just before returning JSX, add guards:

const safeSummary = typeof summary === 'string' ? summary : '';


Render them safely:

{safeIcon ? <View style={styles.icon}>{safeIcon}</View> : null}
{safeSummary ? <Text style={styles.summary}>{safeSummary}</Text> : null}


Do not change layout or styles elsewhere.
